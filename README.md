# APLIKASI PENJUAL MAKANAN SISA

Tugas besar Mata Kuliah Dasar Pemrograman Berorientasi Objek

> Aplikasi yang dibuat untuk menjual kembali makanan yang tidak laku atau tidak terjual untuk kemudian dijual kembali dalam diskonan. Program ini akan memiliki fungsi sebagai marketplace namun dengan sedikit penyesuaian dimana barang yang dijual berada di keadaan diskon yang besarannya berdasarkan dekatnya barang menuju tanggal expired atau kadaluarsa. Program ini juga bisa memberikan opsi untuk membagikan barang selain untuk menjual kembali dengan diskon
